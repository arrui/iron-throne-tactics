# 核心系统设计

> 最后更新：2026-06-06

## 属性体系

### 六项基础属性（所有角色）

| 属性 | 简称 | 作用 |
|------|------|------|
| **武力** | POW | 物理攻击伤害来源 |
| **身法** | SPD | 速差≥5触发追击；影响回避；决定行动顺序 |
| **武艺** | SKL | 命中率 + 暴击率的主要来源 |
| **坚守** | DEF | 物理伤害减免 |
| **命运** | LCK | 暴击率微调、触发概率、被暴击抵抗、冰原存活判定 |
| **体魄** | CON | 携带重型武器不掉速度的能力（承载力） |

> 身法同时管速度和回避，不单开闪避属性。
> 高身法角色天然难打中，符合直觉，通过兵种设计平衡（铁民低速高力，守卫低速高防）。

### 一项特殊属性（仅少数角色拥有）

| 属性 | 简称 | 作用 |
|------|------|------|
| **血脉** | BLD | 魔法攻击力 + 魔法防御力 + 特殊技能触发 |

普通角色属性面板里**没有血脉这一栏**（不是0，是根本不存在），
只有特定角色才会显示，详见 [04-special-mechanics.md](./04-special-mechanics.md)。

## 战斗公式

### 伤害公式

```
物理伤害 = 武力 + 武器攻击力 - 敌方坚守
魔法伤害 = 血脉 + 魔法攻击力 - 敌方血脉防御
最低伤害 = 1（永远不打出0）
```

### 命中率公式

```
攻击方命中 = 武艺×2 + 命运÷2 + 武器命中加成
防御方回避 = 身法×2 + 命运÷2 + 地形回避加成
最终命中率 = 攻击方命中 - 防御方回避
范围：1% ~ 99%（不存在必中或必闪）
```

### 暴击公式

```
暴击率    = 武艺÷2 + 技能加成 - 敌方命运
暴击伤害  = 普通伤害 × 3
```

### 追击条件

```
身法差 ≥ 5 → 追击成立 → 本回合攻击两次
```

## 地形系统

| 地形 | 移动消耗 | 防御加成 | 回避加成 | 备注 |
|------|---------|---------|---------|------|
| 平原 | 1 | 0 | 0 | 标准地形 |
| 森林 | 2 | +10% | +20% | 骑兵消耗×2 |
| 城墙/塔楼 | — | +30% | +10% | 仅步兵可占据 |
| 城门 | 3 | +20% | 0 | 攻防关键节点 |
| 河流 | 3 | -10% | 0 | 铁民不受惩罚 |
| 深雪/冰原 | 3 | 0 | -10% | 非北境兵种消耗×2 |
| 沼泽 | 4 | 0 | -10% | 骑兵无法进入 |
| 废墟 | 2 | +10% | +10% | 君临战役常见 |
| 龙火燃烧地 | 2 | -20% | 0 | 每回合受2点伤害，固定持续3回合后变为烧焦废墟 |
| 烧焦废墟 | 1 | -10% | 0 | 龙火消散后的永久地形，黑色焦土，部分角色有专属反应台词 |

## 武器系统

### 武器耐久度（保留FE经典机制）

- 所有普通武器有耐久度，用完损毁
- 精良武器：耐久度较高（约50次），可修复
- **瓦雷利亚钢**：无耐久消耗，全游戏仅3-5件，各有专属名字和背景故事，不可购买，只能通过剧情获得

### 武器等级

```
E → D → C → B → A → S
与角色熟练度挂钩，使用对应武器积累经验升级
```

### 武器类型（草案）

| 类型 | 特点 | 对应兵种 |
|------|------|---------|
| 单手剑 | 均衡 | 骑士、家臣 |
| 双手剑 | 高攻低速 | 重装步兵 |
| 斧 | 高攻，命中偏低 | 铁民战士 |
| 长枪 | 克骑兵，可反击 | 守卫、长枪兵 |
| 弓 | 远程，无法反击近战 | 游侠、弓手 |
| 匕首 | 高速低攻，毒属性可选 | 影卫、沙蛇 |
| 法杖 | 治疗/诅咒，血脉角色专属 | 红袍祭女、绿先知 |

> 补充系统待设计：商人/铁匠（武器补充来源）、每章武器投放平衡

## 支援系统

- 支援等级：C → B → A（配对系统：A级可进一步到S）
- 条件：相邻行动积累支援点数
- 战场加成：相邻单位提供命中/回避bonus
- 支援对话：揭示角色深度，是叙事的核心载体
- 世代传承：A级以上可建立配对，影响子代属性与技能继承

# 变更日志

## 序章·一完整版（2026-06-07）

### 新增
- **开场动画**：5段旁白幻灯片，黑屏白字淡入淡出，空格/点击可跳过
- **对话系统**：JSON驱动，逐字显示，支持战前/战后对话触发
- **战斗动画**：底部面板弹出，双方图标冲击动画，伤害数字浮动，追击演示
- **FE风格图标**：4个角色像素图标（奈德/劳勃/豪兰/王军）

### 完整游戏流程
1. 开场动画（开场5段旁白）
2. 战前对话（劳勃与奈德的对话）
3. 战斗（含战斗动画面板）
4. 战后对话（胜利后5行对话）
5. 结算面板（重新开始）

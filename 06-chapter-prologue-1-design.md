# 序章·一《风暴地》设计文档

> 最后更新：2026-06-06  
> 状态：原型开发中

## 本章目标

### 教学目标
- 移动单位
- 攻击敌人
- 回合结束
- 战斗预测界面（命中率/伤害预览）

### 情感目标
- 让玩家感受到史塔克/拜拉席恩的仇恨动机
- 建立奈德和劳勃的情感基础
- 基调：沉重，不是胜利的欢呼

---

## 地图设计

### 规格
- 尺寸：15×10格
- 视角：2D俯视，方格地图
- 场景：风暴地山道，夜晚或阴天（体现基调）

### 地形布局草图

```
图例：崖=峭壁(不可进入) 平=平原 林=森林 墙=矮墙
我方出生点：左侧(列0-2)
敌方出生点：右侧(列12-14)
胜利条件：占领右侧营地(列13,行4-5)
```

### 地形说明
- **峭壁**：地图边界，不可进入，形成山道走廊
- **平原**：移动消耗1，无加成
- **森林**：移动消耗2，防御+10%，回避+20%，骑兵消耗×2
- **矮墙**：移动消耗2，防御+20%，仅步兵可通过（骑兵无法翻越）
- **营地**（胜利目标）：占领后触发胜利

## 剧情节拍

### 开场动画（约60-90秒，非可玩）
```
画面1：王城龙穴，疯王坐在铁王座
        "把布兰登·史塔克带上来。"

画面2：布兰登被勒上绞索，瑞卡德跪在地上
        焰火点燃，父子二人的惨叫
        黑屏

画面3：谷地，鹰巢城
        信使："陛下要求交出奈德·史塔克和劳勃·拜拉席恩。"
        琼恩·艾林站起身，沉默片刻
        "不。"

画面4：奈德和劳勃并肩站在山道上，义军旗帜在风中
        旁白："篡夺者战争，第一年。"
```

### 战前对话（可跳过）
```
劳勃：（握紧战锤）"他们欠我们一条命。"
奈德：（看着前方）"不止一条。"
劳勃："打完这仗，我要亲手把那个疯子从王座上拉下来。"
奈德：（沉默，向前走）
```

### 战中事件（可选触发）
```
条件：劳勃与奈德相邻时
劳勃："奈德，左边！"
奈德："我看见了。"
→ 触发支援C级，两人相邻时命中+5%（教学提示：支援加成）
```

### 战后对话
```
营地占领后——
劳勃：（看着升起的义军旗帜）"第一仗。"
奈德："嗯。"
劳勃：（转向奈德）"你在想他们。"
奈德：（停顿）"一直都在想。"
劳勃：（沉默片刻）"我也是。走吧，还有很多仗要打。"
画面淡出，没有胜利音乐，只有风声
```

## 原型开发范围（MVP）

第一个可玩原型只需要实现以下内容，其余全部留空：

### 必须实现（P0）
- [ ] 方格地图渲染（15×10）
- [ ] 单位移动（点击单位→显示可移动范围→点击目标格）
- [ ] 攻击（移动到敌方相邻格→选择攻击→战斗预测弹窗）
- [ ] 战斗结算（伤害计算、HP扣减、死亡判定）
- [ ] 回合切换（我方回合→敌方回合→我方回合）
- [ ] 敌方AI（最简单的：向最近我方单位直线移动并攻击）
- [ ] 胜利/失败条件判定

### 暂不实现（留给后续迭代）
- 剧情对话系统
- 支援系统
- 武器耐久度
- 动画效果
- 音效音乐
- 地形特殊规则（先做平原，森林/墙后加）

## Godot项目结构

```
iron-throne-tactics-godot/
├── project.godot
├── scenes/
│   ├── battle/
│   │   ├── BattleMap.tscn          ← 主战斗场景
│   │   ├── Unit.tscn               ← 单位基础场景
│   │   └── BattlePrediction.tscn  ← 战斗预测弹窗
│   └── ui/
│       └── HUD.tscn                ← 战斗界面
├── scripts/
│   ├── battle/
│   │   ├── BattleMap.gd            ← 地图逻辑、回合管理
│   │   ├── Unit.gd                 ← 单位属性、行动状态
│   │   ├── BattleCalculator.gd     ← 伤害/命中公式（纯计算，无副作用）
│   │   └── EnemyAI.gd              ← 敌方AI
│   └── data/
│       └── UnitData.gd             ← 单位数据结构定义
├── data/
│   └── units/
│       ├── ned_stark.json          ← 奈德属性数据
│       ├── robert_baratheon.json   ← 劳勃属性数据
│       └── howland_reed.json       ← 豪兰属性数据
└── assets/
    ├── tilesets/                   ← 地图图块（先用占位色块）
    └── units/                      ← 单位图标（先用占位色块）
```

# 核心人物立绘重制设计（第一批风格锚点 + 核心扩展）

## 目标

把现有过于抽象、图标化的人物立绘，升级为一套更成熟、庄重、低奇幻、具有战术 RPG 质感的角色肖像系统。

本阶段不直接重制全体人物，而是先建立一套可复用的立绘语言：
- 先用 **奈德·史塔克** 与 **雷加·坦格利安** 两张高分辨率母版做“风格锚点”；
- 锚点确认后，扩展到本批 8 个核心角色；
- 最终导出统一风格的游戏内 `96x96` 成品头像，并替换现有 portrait 资源。

## 本阶段范围

### 风格锚点角色
1. 奈德·史塔克 `ned_stark_portrait.png`
2. 雷加·坦格利安 `rhaegar_targaryen_portrait.png`

### 核心扩展角色
3. 劳勃·拜拉席恩 `robert_baratheon_portrait.png`
4. 霍兰·里德 `howland_reed_portrait.png`
5. 亚瑟·戴恩 `arthur_dayne_portrait.png`

7. 王军指挥官 `royal_guard_captain_portrait.png`
8. 兰尼斯特士兵 `lannister_soldier_portrait.png`

## 不在本阶段内

- 地图块像素精细化
- 全兵种立绘批量重做
- 地图行走小头像 / map sprite 重做
- Dialogue / Deploy / UI 外框结构重设计
- 角色立绘系统逻辑改造

本阶段的重点是：**建立新立绘视觉基线并安全替换现有 portrait 资产。**

## 设计原则

### 1. 形象来源
采用：
- **剧版辨识为主**：五官、发型、胡须、发色、年龄状态尽量贴近剧版记忆点；
- **原著气质强化为辅**：在不偏离剧版辨识的前提下，强化人物的性格、命运感与身份气场。

不是做明星照片复制，也不是做卡通二创；而是做：
**“剧版可识别的高质量策略 RPG 人物立绘”**。

### 2. 风格定位
最终质感采用：
- **折中路线**（非纯影视写实、非纯概念插画）
- 以写实五官为基础
- 加强戏剧化光影、盔甲材质、布料层次、披风和徽记辨识
- 整体更像高端战术 RPG / SRPG 角色肖像

关键词：
- 庄重
- 克制
- 战乱年代感
- 低奇幻
- 金属 / 皮革 / 羊毛 / 披风质感明确
- 非 Q 版、非扁平图标、非儿童绘本风

### 3. 构图规则
统一构图基线：
- **半身或胸像偏上**
- 允许 **3/4 侧脸**，不要求全部正面
- 脸始终是第一识别点
- 肩部、披风、护甲、头盔作为第二识别层
- 背景必须简洁，为人物服务
- 缩到 `96x96` 后，仍需一眼看出是谁 / 属于哪个阵营

### 4. 游戏适配规则
虽然会先做高分辨率母版，但最终目标是游戏头像，因此每张图必须满足：
- 中心主体聚焦明确
- 脸部不要过小
- 头发 / 胡须 / 头盔边缘在小尺寸下不糊成一团
- 明暗对比足够支撑小图识别
- 阵营色只作为辅助手段，不能喧宾夺主

## 角色视觉策略

### 奈德·史塔克
**视觉关键词：** 北境、沉静、疲惫、荣誉、狼一般的克制感  
**方向：** 冷灰蓝调、厚重毛皮/皮革、略显风霜、眼神坚决但不张扬。  
**避免：** 过度年轻、过度热血、王者海报式夸张。  

### 雷加·坦格利安
**视觉关键词：** 王室、悲剧、优雅、忧郁、宿命感  
**方向：** 红黑金结构，银发高贵但不过分仙气，目光内敛，盔甲与王族气场并存。  
**避免：** 太像精灵、太浪漫偶像化、缺乏战争阴影。  

### 劳勃·拜拉席恩
**视觉关键词：** 强壮、粗豪、耗损中的王者、战士余烬  
**方向：** 重体格、厚胡须、暖金棕色调、力量感强但有放纵消耗痕迹。  
**避免：** 仅做成“壮汉模板”。  

### 霍兰·里德
**视觉关键词：** 敏锐、轻捷、湿地、低调、观察者  
**方向：** 轻甲、深绿褐调、略带湿地游猎感，体量小于奈德与劳勃，但精神性更强。  
**避免：** 太普通士兵化、太少年化。  

### 亚瑟·戴恩
**视觉关键词：** 纯白、神圣、锋利、传奇骑士  
**方向：** 浅色甲胄、高贵秩序感、近乎神话般的干净锋利。  
**避免：** 太圣骑士游戏模板化、太浮夸宗教感。  

### 詹姆·兰尼斯特
**视觉关键词：** 金发、傲气、危险贵族、精英骑士  
**方向：** 金红调、姿态自信、五官俊朗但不能柔弱，带一点轻蔑和锋芒。  
**避免：** 纯花美男化、失去战士感。  

### 王军指挥官
**视觉关键词：** 秩序、压迫、王室武装精英、冷硬  
**方向：** 规整头盔/胸甲、制式威权感、比普通士兵更沉重。  
**避免：** 与皇家卫兵普通兵完全同质化。  

### 兰尼斯特士兵
**视觉关键词：** 制式、赤金、纪律、可量产识别  
**方向：** 作为兵种代表，需要比主角更模板化，但仍有更真实的头盔/面甲/甲片结构。  
**避免：** 纯色块兵牌、过度抽象。

## 产出结构

每个角色最终至少对应两层产物：

### A. 高分辨率母版
用途：
- 锁定人物风格
- 作为未来扩展其他 UI 立绘的基底
- 保证缩小后仍有高质量细节支撑

建议：
- 比现有 `96x96` 明显更高的分辨率生成与整理
- 保留原始母版文件，避免以后重复从小图重做

### B. 游戏成品头像
用途：
- 替换当前 `game/冰与火/assets/units/*_portrait.png`
- 保持测试通过
- 保持 Dialogue / Deploy / Battle 中的使用方式不变

硬约束：
- 仍为方形
- 仍满足当前测试要求（至少 `96x96`，实际可保持 `96x96`）
- 不改文件名，不改映射名

## 风格控制方法

为了避免 8 张图“各自好看但不在同一个游戏里”，生成与整理时必须统一这些维度：

### 1. 光影
- 主光源方向尽量统一
- 面部与肩甲需要有明确层次
- 避免一张偏雾面、一张偏硬闪、一张偏棚拍的混乱情况

### 2. 背景
- 背景只能做低复杂度氛围衬托
- 不允许抢人物主体
- 不做复杂场景叙事背景
- 背景颜色服务于人物身份与阵营识别

### 3. 颜色
- 奈德 / 霍兰偏冷
- 雷加 / 詹姆 / 兰尼斯特士兵偏暖红金
- 亚瑟偏浅银白
- 劳勃偏暖棕金
- 王军指挥官偏冷硬制式金属

### 4. 材质
必须真正拉开：
- 毛皮
- 皮革
- 锁甲
- 抛光板甲
- 披风/羊毛/织物

### 5. 小图可读性
所有大图都要接受“小图裁切复核”：
- 眼睛、鼻梁、胡须、头盔、发际线是否仍清楚
- 高光是否变成噪点
- 背景是否吞掉头部轮廓

## 风险与对策

### 风险 1：太贴剧照，失去游戏立绘感
**对策：** 不做照片复刻，统一加绘画化光影与材质整理。  

### 风险 2：太绘画化，失去剧版辨识
**对策：** 核心面部特征以剧版记忆点为第一优先。  

### 风险 3：大图很好，小图糊掉
**对策：** 每张角色都必须经过 `96x96` 小图复核。  

### 风险 4：兵种与主角风格断裂
**对策：** 兰尼斯特士兵和王军指挥官也按同一光影与材质语法生成，只是角色个性弱于主角。

## 建议的实现边界

为了避免此阶段失控，建议这轮只做：
- 核心人物立绘母版与游戏头像导出
- 必要的资源替换与验证

不要在这一轮顺手做：
- map sprite 重绘
- UI 相框风格大改
- 全角色立绘扩编
- 地图像素精修并行推进

这样能保证本阶段作为“人物立绘重制第一段”独立收口。

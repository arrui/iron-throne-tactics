# 叙事结构与主线设计

> 最后更新：2026-06-06

## 序章结构（共4章）

### 序章第一章：《钢与誓言》
**时间：** 篡夺者战争中期，奈德约18岁  
**教学目标：** 移动、攻击、回合结束、战斗预测界面  
**地图：** 山道，约15×10格（峭壁封边，中段矮墙，终点小营地）  
**我方：** 奈德（剑士）、劳勃（重装）、豪兰·里德（长枪）  
**敌军：** 5名疯王士兵，无特殊行为，直线冲锋  

**剧情节拍：**
- 战前：三人轻松对话，展现年轻时的义气
- 战后：营地里找到一封信，得知疯王杀了布兰登和瑞卡德
- 动画：疯王宝座前的焰火，奈德沉默，劳勃愤怒

---

### 序章第二章：《三叉戟》
**时间：** 篡夺者战争决战，三叉戟河畔  
**教学目标：** 支援系统、兵种克制初体验  
**地图：** 河畔战场，约20×15格（河流地形、浅滩渡口、开阔平原）  
**我方：** 劳勃（主角，重装战锤）、奈德（支援）、其他义军将领  
**敌军：** 雷加·坦格利安（Boss）及王军  

**设计关键：**
- 雷加是本游戏第一个龙骑士Boss形态的预演（此时无龙，但展现坦格利安精锐）
- 劳勃击败雷加的那一击做成全屏演出，战锤对胸甲，红宝石四散
- 战后劳勃重伤，奈德分兵——劳勃去君临，奈德南下极乐塔

**剧情节拍：**
- 战前：劳勃怒吼"还我莱安娜"，全军士气点燃
- 战中Boss击败演出：雷加倒下，红宝石散落河中
- 战后：劳勃重伤，奈德："你去君临，极乐塔我去。"两人分兵

---

### 序章第二章续：《极乐塔》
**时间：** 三叉戟之战后数日，多恩  
**教学目标：** 地形回避、攻坚战、胜利条件不等于击败所有敌人  
**地图：** 多恩沙漠，一座塔楼，约18×14格  
**我方：** 奈德（主角）、豪兰·里德、少量北境骑士  
**敌军：** 亚瑟·戴恩（Boss，极强）、多恩骑士  

**亚瑟·戴恩的处理：**
- 设计为"不可正面击败的障碍"——他的属性远超玩家当前等级
- 胜利条件：奈德抵达塔楼顶端（不是击败戴恩）
- 豪兰·里德的侧翼行动作为剧情触发条件（非玩家操作），戴恩倒下

**剧情节拍：**
- 战前：奈德只说"莱安娜在里面"，没有多余的话
- 战中：艰难推进，戴恩如同不可逾越的墙
- 触发：奈德抵达塔楼门口，过场——豪兰从侧面刺入，戴恩倒下
- 奈德独自进入塔楼（玩家操控结束）
- 动画：昏暗室内，莱安娜的声音："奈德……答应我……"黑屏，婴儿哭声，不展示孩子的脸
- 战后：奈德从塔楼走出，怀里抱着什么，豪兰在旁，两人沉默无言

---

### 序章第三章：《铁王座》
**时间：** 君临陷落当日  
**教学目标：** 章节准备画面、武器耐久、道具使用、大规模地图部署  
**地图：** 君临城内，约25×18格（街道、广场、王座大厅外庭）  
**我方：** 奈德（主角）、北境军队  
**敌军：** 残余王军（溃散状态，命中率惩罚）、精锐近卫（Boss）  
**中立单位：** 兰尼斯特军（泰温的军队，不攻击我方但也不帮助，可见其在城内屠杀）  

**关键人物出场：**
- **詹姆·兰尼斯特**：奈德推进到王座大厅，詹姆坐在铁王座台阶上，疯王尸体在旁，两人对峙（过场，不可玩）。詹姆的道德复杂性此刻埋下，不展开，留给罗柏线
- **泰温·兰尼斯特**：地图上可见兰尼斯特军在城内行动，泰温的旗帜，暗示"盟友"的真实面目
- **劳勃·拜拉席恩**：最后抵达，战后过场出现，坐上铁王座

**剧情节拍：**
- 战前准备画面：首次完整体验部署、武器状态、道具栏
- 战斗：推进穿越混乱的君临街道，中立的兰尼斯特军令人不安
- 抵达王座大厅：詹姆过场，奈德的沉默比任何台词都有力
- 战后：劳勃抵达，坐上铁王座，奈德站在一旁
  - 劳勃："詹姆的事……算了。"
  - 奈德皱眉，没有说话
  - 劳勃："奈德，留下来。做我的首相。"
  - 奈德："我要回北境了。"
- 最后画面：临冬城，雪中，年幼的罗柏从城门跑出迎接父亲，奈德抱起他，身后跟着一个年龄相仿的黑发男孩（背影，不给正脸）
- 旁白："多年之后，他的儿子将会面对一个更残酷的世界。"
- 黑屏，标题：《铁王座战记》，时间跳转提示：【二十年后】

## 整体叙事框架

```
序章：篡夺者战争（奈德·史塔克视角）
    约2-3章，战棋教学，建立史塔克家族情感根基
    结束于：年轻的奈德凯旋，但战争代价已埋下

        ↓

第一幕：五王之战（罗柏·史塔克视角）
    约6-7章，玩家在这里真正掌握战棋系统
    建立对罗柏麾下将士的感情
    结束于：红色婚礼，硬切黑屏
    ← 全游戏最黑暗的时刻 →

        ↓

第二幕：分裂（双线并行）
├── 乔恩线：约5章，长城→极北之地
│   带着对红色婚礼的愤怒进入
│
└── 艾莉亚线：约4章，流亡→布拉佛斯→归途
    切换视角，情感落差极大

        ↓

第三幕：归途（双线汇合）
    约3章，乔恩血脉揭露，长城以北白鬼大军
    结束于：凛冬已至，留下续篇空间
```

## 核心叙事决策

| 决策 | 结论 |
|------|------|
| 红色婚礼 | 忠于原著，罗柏必死，不设拯救条件 |
| 第一幕终点 | 长城告急/白鬼威胁初现，引出雪诺线 |
| 第二幕结构 | 乔恩·雪诺 + 艾莉亚·史塔克双主线，参考烈火之剑双主角 |

## 关于"已知结局"的叙事立场

玩家知道罗柏会死——这不是问题，而是设计资产。
原著读者第一次读红色婚礼是**震惊**，游戏玩家经历红色婚礼是**心碎**，这是两种不同的情感体验，都有价值。
玩家在罗柏章节学会战棋，建立对角色的感情，然后这一切在红色婚礼被打碎——
带着这份愤怒和悲伤进入乔恩/艾莉亚的章节，这是极好的叙事引擎。

## 双主线对比

| 维度 | 乔恩·雪诺线 | 艾莉亚·史塔克线 |
|------|-----------|---------------|
| 地理 | 长城 → 极北之地 | 流亡 → 布拉佛斯 → 归途 |
| 战场风格 | 开阔，大规模战役 | 狭窄，暗杀/小队突袭 |
| 主题 | 身份认同、血脉、责任 | 复仇、失去、重新成为自己 |
| 兵种 | 重步兵、游侠、夜巡 | 刺客、流亡者、无脸人 |
| 核心机制 | 隐性血脉觉醒 | 变脸机制 |
| 结局走向 | 白鬼威胁，世界末日感 | 归家，与乔恩线汇合 |

## 艾莉亚专属机制：变脸

```
【变脸】艾莉亚专属技能
─────────────────────────────────────
触发条件：击败特定敌方"命名单位"后，
          艾莉亚可以"记住"该角色的面孔

使用方式：章节准备阶段，艾莉亚选择当前持有的某张"面孔"出战

效果：
  → 以该面孔外观登场
  → 敌方单位不会主动攻击艾莉亚（认为她是自己人）
  → 特定对话选项改变（可接近原本无法接触的角色）

代价：
  → 以他人面孔出战时，艾莉亚无法使用个人专属技能
  → 部分章节有"识破条件"，被识破立即触发警报状态
```

## 乔恩线核心机制：誓言与血脉

```
【誓言选择】某些章节触发
  选择A：遵守夜巡誓言 → 某些同伴离开，但长城防线更稳固
  选择B：回应血脉召唤 → 获得额外资源，但夜巡内部关系恶化

【血脉觉醒】特定章节触发
  隐性坦格利安血脉觉醒
  属性面板出现血脉栏
  解锁与龙相关的剧情可能性（为第三幕汇合埋下伏笔）
```

## 序章角色名单

**动画交代（非可玩）：**
- 父亲瑞卡德·史塔克被疯王活活烧死
- 哥哥布兰登·史塔克被勒死
- 莱安娜·史塔克的死亡（塔楼之乔伊，奈德进门瞬间，不展示孩子，留给乔恩线揭露）

**可玩角色（约5人）：**

| 角色 | 身份 | 序章作用 |
|------|------|---------|
| **奈德·史塔克** | 主角 | 教学关卡，建立史塔克情感基础 |
| **劳勃·拜拉席恩** | 义兄弟 | 并肩作战，为后来落魄胖王埋下对比伏笔 |
| **年轻的巴里斯坦·赛尔弥** | 王卫（敌→中立） | 以"无法击败的敌人"形式出现，展现王卫荣耀 |
| **豪兰·里德** | 奈德挚友 | 塔楼之乔伊关键人物，乔恩身世的伏笔载体 |
| **小罗柏** | 幼儿，不可操控 | 序章末尾抱着婴儿罗柏，为正传主角埋下情感锚点 |

**豪兰·里德人物备注：**
沼泽之地里德家族领主，奈德最忠诚的老友。塔楼之乔伊战役的唯一幸存目击者——他救了被亚瑟·戴恩压制的奈德，也是乔恩·雪诺真实身份的唯一活着的人证。整部原著他几乎未曾现身，是一根埋藏极深的引线。

## 第一幕角色名单（罗柏线）

| 角色 | 身份 | 剧情作用 | 命运 |
|------|------|---------|------|
| **罗柏·史塔克** | 主角 | 从少年到北境之王的成长弧 | 红色婚礼死亡（剧情强制） |
| **凯特琳·史塔克** | 母亲/军事顾问 | 情感核心，判断失误推动悲剧 | 红色婚礼死亡（剧情强制） |
| **詹姆·兰尼斯特** | 俘虏→复杂角色 | 骑士精神的两面，奈德之后第二个道德镜子 | 被凯特琳释放，离开 |
| **黑鱼·徒利** | 将领 | 军事支柱，正直的代表 | 存活（红色婚礼前离场），可延续至第二幕 |
| **多浮·费里** | 表面盟友/主谋 | 全程可疑，红色婚礼策划者 | 叛变，敌方Boss |
| **葛瑞渥·弗雷** | 表面盟友 | 渡口誓言的代价，弗雷家族代表 | 叛变，红色婚礼执行者 |
| **泰温·兰尼斯特** | 主要敌方 | 战场上真正的对手 | 敌方，存活 |
| **瑟曦·兰尼斯特** | 远程敌方 | 通过情报/事件展现，幕后威胁 | 敌方，存活 |
| **罗斯·波顿** | 可疑盟友 | 背叛伏笔（波顿家族） | 可选叛变节点 |
| **罗柏的妻子** | 情感角色 | 渡口誓言代价，红色婚礼导火索 | 红色婚礼死亡 |

**黑鱼·徒利人物备注：**
凯特琳的舅舅，布莱登·徒利，外号"黑鱼"。家族异类，一辈子未婚，独来独往，但极出色的将领。急流城陷落时唯一逃脱者。红色婚礼时恰好去巡逻而幸免。代表徒利家族的骨气——所有人都投降时他还在打。

**多浮·费里人物备注：**
河间地费里城堡领主，"晚宴领主"。表面忠诚，实为全书最工于心计的墙头草之一。红色婚礼主谋之一，向弗雷家族出卖罗柏，换取兰尼斯特庇护和更多领地。也是间接害死奈德·史塔克的幕后推手。凯特琳最痛恨的人。

## 第二幕乔恩线角色名单

| 角色 | 身份 | 剧情作用 | 机制特色 |
|------|------|---------|---------|
| **乔恩·雪诺** | 主角 | 隐性坦格利安血脉，身份认同核心 | 血脉觉醒事件 |
| **火吻·伊葛蕾特** | 野人首领 | 乔恩与野人世界的桥梁，最重要的支援A级角色 | 弓手兵种，高速高武艺 |
| **托蒙德·巨人之妻** | 野人战士首领 | 情感担当，力量与忠诚的代表 | 重型斧手，高武力高体魄 |
| **山姆威尔·塔利** | 夜巡学士 | 知识型辅助，关键时刻不退缩 | 辅助兵种，"学识"被动技能（可触发关卡隐藏信息） |
| **幽灵** | 冰原狼 | 特殊单位，不占部署位，侦察/协助 | 机制待设计 |

**山姆威尔设计备注：**
不拥有血脉属性，但有专属被动"学识"——对古老知识的直觉性理解，在特定关卡触发隐藏信息或特殊对话，是知识而非魔法。

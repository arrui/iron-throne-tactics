# 特殊机制设计

> 最后更新：2026-06-06

## 血脉系统

### 显性与隐性双轨

```
显性血脉角色                      隐性血脉角色
────────────────                 ────────────────────
招募时属性面板直接显示血脉栏         属性面板无血脉栏（和普通角色完全一样）
玩家从一开始就知道他特殊             特定事件触发 → 血脉觉醒
例：梅丽珊卓、丹妮莉丝              面板上突然出现血脉一栏
                                    玩家："原来他一直是……"
```

### 血脉觉醒触发条件

| 触发方式 | 叙事含义 | 举例 |
|---------|---------|------|
| 与特定角色支援达到A级 | 另一个人认出了他的血脉 | 老师傅认出剑法里的家族印记 |
| 在特定地点战斗 | 故土激活了沉睡的血脉 | 踏上龙石岛时觉醒 |
| 主线剧情节点 | 剧情揭露真相 | 特定章节战后过场动画 |
| 角色死亡前的遗言 | 临终告白 | 倒下的父亲说出秘密 |

### 血脉觉醒后的效果

- 属性面板新增血脉栏（初始值偏低，有成长空间）
- 解锁该角色的隐藏支援对话
- 若为坦格利安血脉：解锁驯龙资格
- 副作用：玩家开始猜测每个角色是否有隐性血脉，增加代入感

### 拥有血脉属性的角色类型

| 类型 | 血脉来源 | 能力方向 |
|------|---------|---------|
| 红袍祭女/乌鸦祭司 | 光之王信仰 | 治疗、诅咒、复活 |
| 坦格利安龙骑士 | 龙族血脉 | 驭龙、龙火抗性 |
| 异鬼/夜王 | 死亡魔法 | 复活尸体，免疫物理 |
| 绿先知 | 旧神恩赐 | 预言、树心石、远视 |
| 影子巫师 | 暗影魔法 | 隐身、暗杀增强 |

## 龙与坦格利安机制

### 龙的状态机

```
野生（中立）
    ↓ 坦格利安角色相邻行动X回合，或特定剧情触发
已驯服（绑定）── 成为该坦格利安角色的专属战斗单位
```

- 同一个坦格利安角色只能绑定一条龙
- 驯服需要时间和章节资源投入
- 击杀野生龙可获得龙骨材料（稀有），可制作强力装备

### 龙被击杀后：坠落机制

龙与骑士是一体战斗单位，龙死则判定骑士坠落，结果取决于**坠落地形**。

| 坠落地形 | 骑士结果 | 理由 |
|---------|---------|------|
| **森林** | 存活，HP剩30%，获得【擦伤】 | 树冠缓冲，枝干划伤 |
| **平原** | 存活，HP剩15%，获得【重伤】 | 硬地面重摔，无额外危险 |
| **废墟/城内** | 存活，HP剩20%，获得【压伤】 | 瓦砾缓冲，碎石砸伤 |
| **浅水/河流** | 存活，HP剩25%，获得【溺水】 | 水面缓冲，重甲会下沉 |
| **城墙/高台** | **永久死亡** | 坠落高度叠加，无缓冲 |
| **沼泽** | **永久死亡** | 重甲骑士沉入沼泽，无法自救 |
| **冰原/深雪** | 命运≥10存活（HP剩5%），命运<10永久死亡 | 雪地可能缓冲，也可能冻死 |
| **海洋/深水** | **永久死亡** | 全甲入水，沉没 |
| **龙火燃烧地** | **永久死亡** | 坠入火海 |

### 坠落状态说明

| 状态 | 效果 | 持续 |
|------|------|------|
| **擦伤** | 武艺-3 | 3回合 |
| **重伤** | 身法-4，坚守-3 | 5回合 |
| **压伤** | 体魄-5，武力-2 | 4回合 |
| **溺水** | 身法-5 | 2回合后自动解除 |

### 丧龙之怒（与坠落状态叠加）

```
骑士存活 + 龙被击杀 → 同时获得：
  【丧龙之怒】攻击+20%，防御-10%，持续3回合
  + 对应地形的坠落状态

两种状态叠加 = 极度危险但极度致命
→ 重伤+丧龙之怒的骑士：打起来很疼，但一击即溃
→ ASOIAF里"绝境之人"的感觉
```

### 骑士死亡对龙的影响（反向）

```
骑士永久死亡（战场死亡，非坠龙）
→ 龙失去绑定，重新变为野生中立状态
→ 不主动攻击任何人，也不受任何人控制
→ 可被其他坦格利安角色重新驯服
  （所需时间更长，因为"龙在悲伤"）
```

### 击杀敌方龙的奖励

```
击杀敌方龙 → 龙骨材料（稀有）+ 大量经验值
             敌方龙骑士（若存活）立即获得【丧龙之怒】
```

## 绿先知机制

绿先知**不参与战斗，但影响战局**。

### 预言系统

```
每章开始前，绿先知角色给出一段模糊的预言台词
→ 这段台词是当前章节某个隐藏机制的提示
→ 看懂了可以更好地完成隐藏目标
→ 看不懂也不影响通关，但事后回想会觉得"好准"

例：
"我看见铁树开花，海浪染红了雪。"
→ 暗示：这章有铁群岛角色会在河流地形遭遇危机
→ 提前布置长枪兵可以避免该角色死亡
→ 保护成功：触发隐藏对话
```

### 树心石机制

```
绿先知作为特殊驻守单位：
  - 不能移动，不能攻击
  - 每回合可激活地图上一个"树心石"节点

激活节点效果（二选一）：
  A. 该节点周围3格获得迷雾效果（遮蔽敌方视野）
  B. 触发"看见未来"——提前显示敌方下一回合的移动路径
```

### 代价设计

```
绿先知必须存活 → 预言和树心石能力才能使用
→ 保护他是战术压力，而不是战斗力来源
→ 符合原著里布兰需要保护者的设定
```

## 龙火地形详细规则

```
【龙火燃烧地】
触发：龙骑士使用龙火攻击命中地形单位后，该格变为龙火燃烧地
持续：固定3回合（可预期，玩家可以计划）
每回合效果：身处其中的单位受2点固定伤害（无视防御）
视觉：第1回合烈焰，第2回合余焰，第3回合灰烬闪烁

【消散后变为：烧焦废墟】
防御加成：-10%
回避加成：0
移动消耗：1（正常，但视觉为黑色焦土）
特殊：部分角色有专属台词（看见故土被焚毁时的反应）
持续：永久（该章内不会恢复）
```

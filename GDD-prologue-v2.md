## 一、创意总纲

### 核心定位

**不是"冰火皮肤的火焰纹章"——是"用战旗机制说出冰火灵魂的悲剧"**

圣战系谱留给我们最重要的一课：当玩家爱上了一支军队，你把他们全部杀死的那一章，才是真正的叙事高潮。冰与火之歌天生具备这种残忍的叙事力量。序章要做的，是在4个章节内，让玩家真正*在乎*奈德·史塔克——然后让铁王座大殿的结局，如同第五章的贝尔哈拉屠杀一样刻骨铭心。

### 三个设计支柱

```
1. 荣耀的代价（Honor Has a Cost）
   → 奈德的Honor不是美德，是枷锁
   → 游戏机制要体现：做正确的事，往往要付出更高战术代价

2. 权力的腐蚀（Power Corrupts the Map）
   → 地图越往后，"友军"越不可靠
   → 支援度系统与背叛风险并存

3. 北方记忆（The North Remembers）
   → 过场/对话保留原著的克制与留白
   → 角色台词：说一句藏三句，不点破的才是重量
```

### 参考作品矩阵

| 维度 | 参考作品 | 具体借鉴点 |
|------|---------|-----------|
| 战斗系统 | FE7《烈火之剑》Lyn线 | 10章渐进式教学，每章只引入1-2个机制 |

## 二、美术风格圣经（Art Bible）

### 2.1 整体基调

**拒绝：** Q版、明亮色系、日式RPG少年感
**接受：** 沉重、风化、蜡烛光下的铁锈

参照色盘：

```
主色调：
  铁灰 #3D3D3D  ← 城墙、盔甲、石板路
  深红 #8B1A1A  ← 兰尼斯特纹章、战场血迹
  北绿 #2D4A22  ← 史塔克军，树林
  烛珀 #C8842A  ← UI高亮、对话框、蜡烛光

禁用：
  纯白、纯黑、荧光色、饱和度>60%的亮色
```

### 2.2 角色立绘规范（FE GBA式像素立绘）

参照GBA火焰纹章肖像规格，但**加暗加重**：

```
技术规格：
  分辨率：176×112 像素（FE GBA原版80×80放大到此规格）
  色板：每角色最多16色（含阴影#282828和背景色）
  姿势：上半身¾侧面，带武器或道具
  背景：单色暗调（不同阵营用不同背景色）

构图风格：
  史塔克阵营：背景深蓝灰，沉稳
  兰尼斯特阵营：背景深红褐，威压
  皇家卫队：背景铁灰，冰冷
  中立/中间人：背景橄榄暗绿
```

### 2.3 地图精灵（Map Sprite）规范

```
单位精灵：32×32像素
  → 形体可辨认（头盔轮廓 + 武器）
  → 移动动画：2帧来回（左右脚交替）
  → 受击动画：向后退1格（1帧）
  → 阵亡动画：倒地（2帧淡出）

地形瓦片：72×72像素（当前项目已设定）
  → 每种地形3个变体（防止重复感）
  → 城市道路：青石板，有裂缝纹理
  → 红堡外墙：巨石砌成，每隔5格有箭垛
  → 内殿地板：人字形拼接石砖，暗色
```

### 2.4 UI设计风格

```
对话框：
  边框：做旧羊皮纸+金属铆钉角
  字体：衬线体，偏书法感（非圆体）
  角色名：压在左上角金色印章内

地图UI：
  回合指示：左上角烛光图标+文字
  HP条：铁锈红填充，背景黑
  行动点：铸剑蓝填充

战斗预测弹窗：
  简洁卷轴样式
  攻方数据左，守方数据右
  伤害数字：粗体红色
```

### 序章一：「北境来客」

**主题：** 荣耀降临一座腐败的城市

**地图规格：** 10×8 格（小地图，聚焦教学）

**胜利条件：** 奈德抵达红堡正门（有标记）

**敌军配置：**
```
3名皇家城卫兵（刷新位置：正门左右各1，远端1）
※ 第一章敌兵HP极低（12HP），确保新手1-2击可击倒
```

**教学节点时序：**

```
[进图触发] → 对话框：小指头出现，礼貌介绍
   ↓ 关闭对话
[弹窗01] "北境总督，现在是您的回合。
          点击您的单位选中它——"
   ↓ 强制高亮奈德格
[玩家点击奈德] 
[弹窗02] "蓝色范围是您可以移动的格子。
          将奈德移动到您想要的位置。"
   ↓ 玩家自由移动
[移动后/若接近敌人]
[弹窗03] "红色格子内有敌人。点击红格发动攻击。
          攻击前会显示战斗预测——"
   ↓ 强制打开战斗预测弹窗，暂停等玩家阅读
[弹窗04] "命中率是攻击成功的概率。
          按【确认攻击】发动战斗。"
   ↓ 玩家确认攻击，播放战斗动画
[击倒第一个敌人后]
[弹窗05] "干得漂亮。回合结束后，敌人会反击。
          按右下角【结束回合】按钮——"
   ↓ 等待玩家点击结束回合
[敌方回合结束后]
[弹窗06] "地图上的星形标记是胜利目标。
          引导您的单位到达那里完成任务。"
```

**剧情对话（重写版）：**

> 【奈德·史塔克，抵达君临城第3日】

**奈德：** 这座城的石头都不对劲。
**凯特琳（回忆/旁白）：** 你总是不信任南方的一切，奈德。
**奈德：** 凯特，这里的空气都带着谎言的味道。

---

*【皇家卫兵拦住去路】*

**卫兵长：** 阁下须接受例行检查——这是陛下的命令。
**奈德（内心）：** 凡人守则，我都遵从。但有些规矩，我不会接受。
**奈德：** 我是北境总督，奉国王之命赴任。让开。
**卫兵长：** 手令呢？

*【奈德用手按上冰之泪的剑柄，没有拔剑】*

**奈德：** ……你这样想让国王知道吗？

*【卫兵迟疑，让开】*

**小指头（在一旁观察）：** 呵。北境来的熊——直接用獠牙开路。有趣，有趣。

---

**章节后过场（胜利后）：**

> 奈德站在红堡正门前，望着那座用铁与血建成的城堡。
> 他想起父亲，想起他的哥哥，想起那把燃烧的铁王座传说——
> 然后他想起罗伯特说的话：*"只有你，奈德，我才信任。"*
> 
> 他深吸一口气，走进去。
> 没有人告诉他，这扇门比他想象的难走出去。

### 序章二：「小指头的礼物」

**主题：** 在正确与有利之间，奈德选择正确

**地图规格：** 16×12 格

**胜利条件：** 击败敌方指挥官 OR 奈德抵达证据所在格

**新机制引入：**

```
1. 地形加成系统（第一次讲解）
   → 兰尼斯特区域有走廊，进入走廊+20回避
   → 弹窗说明：地形图标含义

2. 等待行动（Wait）
   → 有时不攻击，等待更好时机更有利

3. 【荣耀系统首次提示】
   → 当玩家可以追击已经投降/跪地的士兵时
   → 弹窗提示："奈德总督不会斩杀请降之人。
                  此格行动被史塔克家徽约束。"
   → 玩家自动跳过这次攻击
```

**核心剧情：**

> 小指头"好意"给奈德提供了情报路线，但奈德发现这条路上有陷阱——他需要绕远路，代价是损耗更多行动力，却能找到更重要的证据。

**关键对话：**

**培提尔·贝利席（小指头）：** 总督大人，有条捷径，让我的人带您去——
**奈德（直视他眼睛）：** 你的人。
**贝利席：** 当然，我对您绝无——
**奈德：** 培提尔。我们认识多少年了？
**贝利席（停顿）：** ……足够久。
**奈德：** 所以你知道，我不走捷径。

*【奈德转身选择另一条路，贝利席目送他离开，微微一笑——那种微笑】*

---

### 序章三：「黄金披风的背刺」

**主题：** 最坚固的盟约，被最卑微的人破碎

**地图规格：** 20×16 格

**胜利条件：** 奈德护送指定角色到达出口

**核心机制——背叛触发：**

```
【回合5开始前】
  → 地图上有8名"黄金披风城卫"作为友军（绿色）
  → 回合5开始，触发过场对话
  → 小话框：扬诺斯·史林特收到一枚金币
  → 所有黄金披风城卫变色：绿 → 红（转为敌方）
  → 弹窗警告："背叛！城卫军倒戈——重新评估战场！"

设计意图：
  玩家之前可能依赖城卫军的保护
  背叛后，战场突然对玩家不利
  强迫玩家重新思考位置和出口
```

**关键对话（背叛前夕）：**

**奈德（对雷克·塔里）：** 我需要你带孩子们先走。
**塔里：** 总督，我不能——
**奈德：** 这是命令，雷克。**带她们回北境。**
**塔里（沉默片刻）：** ……遵命，总督。

*【塔里离开。奈德独自面向城堡，背对观众。】*

**旁白（奈德内心）：** 我应该告诉罗伯特。我应该，在三天前，就告诉他真相。
**旁白：** 但我想给他的孩子一条活路。
**旁白：** 也许这就是我的错。

---

*【回合5——扬诺斯·史林特出现】*

**史林特：** 总督大人……很抱歉。
**奈德：** 史林特。
**史林特：** 王后有令——
**奈德：** 你卖掉了你的荣耀，换了什么？黄金？
**史林特（没有回答，转向士兵）：** 拿下他。

---

### 序章四：「铁王座之下」

**主题：** 有些战争，不是用剑赢得的

**地图规格：** 24×18 格（最大地图）

**胜利条件：** 奈德独自（不杀任何人）到达铁王座格

**失败条件：** 奈德阵亡

**特殊机制——铁王座的重量：**

```
奈德抵达铁王座格后，不触发即时胜利
而是触发最终过场对话，玩家只能【等待】

这一章没有"打赢"的出路
剧情设计为：奈德选择放下武器，相信小乔弗里
（即原著中他做出的选择）

游戏在这里做出明确的叙事表态：
"您已经做到了您能做到的一切。
 荣耀是奈德的剑，也是他的枷。"
```

**结局过场（参照FE4贝尔哈拉屠杀的震撼感）：**

```
画面：铁王座大殿
奈德单人立于王座前，身后是空荡荡的宫殿

[文字缓缓出现，不配任何音效]

"他本可以逃走。"
"他本可以妥协。"
"他本可以说谎。"

[停顿4秒]

"但他是奈德·史塔克。"

[画面黑屏，白字]
"北境的冬天即将来临。"

[远处传来狼嚎——幻听般的音效]

[出现序章完成界面：
  "序章完毕
   真正的故事，从此开始"]
```

## 四、玩法引导设计（Tutorial Blueprint）

### 引导总原则（参照FE7 Lyn线）

```
1. 每章只引入1-2个新机制，绝不堆砌
2. 引导弹窗：出现 → 等待玩家操作 → 自动关闭（不阻断游戏流程）
3. "看到就能学会"：UI自己会说话，弹窗只强调重点
4. 第三章开始，撤除所有弹窗提示，玩家自主作战
```

### 机制引入时序表

| 章节 | 新引入机制 | 隐藏机制（玩家自己发现） |
|------|-----------|----------------------|
| 序章一 | 移动、基础攻击、结束回合 | 地形颜色暗示 |
| 序章二 | 地形加成、等待行动、荣耀系统 | 多单位协作加成 |
| 序章三 | 支援系统、危险区（D键） | 背叛事件 |
| 序章四 | 无（纯叙事关卡） | 铁王座机制 |

### 具体弹窗文案规范

```
弹窗要求：
  - 每条不超过35字（手机单行可显示）
  - 用第二人称（"您的单位"而非"玩家"）
  - 不用"注意"、"重要"等词——直接说要做什么
  - 操作说明后面跟一句感受性描述

示例：
  ❌ "注意：请点击蓝色范围内的格子移动单位"
  ✅ "点击蓝色格子，带奈德走向他的命运。"

  ❌ "地形加成可以提高防御和回避"
  ✅ "石柱后面的人更难被击中。把奈德放在掩护处。"
```

## 五、角色声音手册（Character Voice Guide）

### 奈德·史塔克

**原著声音关键词：** 笨拙于政治、诚实到自伤、爱家人超过爱自己

```
说话方式：
  → 短句为主，很少使用长从句
  → 直接陈述，不打比喻，不拐弯
  → 愤怒时：更安静，不是更大声
  → 内心独白：充满疑虑，但行动果断

禁用词汇：
  → "正义"（他不说大词）
  → "为了……而战"（他只是做他觉得对的事）
  → "相信我"（他的行动说话，不是嘴）

示例对比：
  ❌（AI味）："身为北境总督，我必须为正义而战，扞卫王国的法统！"
  ✅（原著味）："我得告诉罗伯特真相。就这样。"

  ❌："我已经决定了，不管付出什么代价！"
  ✅："我想错了什么？……不，我没有想错。只是来晚了。"
```

### 培提尔·贝利席（小指头）

**原著声音关键词：** 一切都是买卖，连感情都标价

```
说话方式：
  → 永远礼貌，永远含一个问句
  → 给人台阶下，然后在台阶上装地雷
  → 用第一人称时是"我们"，不是"我"
  → 说完之后，总有一个停顿——让对方先说

禁用：
  → 直接表明立场
  → 急迫感

示例：
  ❌："总督大人，我是您的朋友，请相信我！"
  ✅："总督大人……当然，您有您的判断。
       我只是想说——这条路，我走过很多次了。"
  [停顿]
  ✅："不过，史塔克家的人向来有自己的路。"
```

### 瑟曦·兰尼斯特

**原著声音关键词：** 从不道歉，因为她从未认为自己错了

```
说话方式：
  → 命令句，即使是请求，也是命令的形状
  → 每句话都在确认自己的权力位置
  → 提到"孩子"时，声音会柔化一层——但不失控

示例：
  ❌："奈德总督，请您理解我的处境——"
  ✅："总督。我的孩子不会因为您的荣耀而受苦。"

  ❌："这是我的命令！"
  ✅："这，是最后一次机会。"
```

### 詹姆·兰尼斯特

**原著声音关键词：** 他比你聪明，他也比你痛苦，他恨自己知道这两件事

```
说话方式：
  → 大量反讽，但底层是悲伤
  → 提到"弑君"时，平静得过于刻意
  → 对奈德有特殊的尊重——两人都是无法融入南方的人
  → 笑声出现在最不该笑的时候

示例：
  ❌："哈哈，奈德史塔克，看看你现在的处境！"
  ✅："总督大人。" [看着被围困的奈德，停顿]
      "你有没有想过，我们两个……其实挺像的？"
  [奈德没有回答]
  ✅："都是好人。" [詹姆转过脸去] "好人总是输的早。"
```
